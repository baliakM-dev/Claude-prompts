# Enterprise Code Review

Review the proposed or changed code as a principal engineer for a regulated enterprise environment.

Check the following:
1. Security
2. SOLID principles
3. Architecture alignment
4. Transaction boundaries
5. API contract stability
6. Persistence design quality
7. Logging and observability
8. Test coverage adequacy
9. Backward compatibility
10. Production readiness

Return:
- executive summary
- critical issues
- important improvements
- risks
- merge readiness verdict
